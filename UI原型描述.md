# UI界面原型描述

## 1. 设计规范

### 1.1 色彩体系

| 用途 | 颜色名称 | 色值 | 说明 |
|------|----------|------|------|
| 主色 | 声乐蓝 | #2E6BB0 | 主按钮、导航高亮 |
| 辅色 | 优雅紫 | #8B5CF6 | AI功能、趋势图表 |
| 强调 | 活力橙 | #F59E0B | 评分、徽章 |
| 成功 | 翠绿 | #10B981 | 正常、改善 |
| 警告 | 琥珀 | #F59E0B | 一般、提醒 |
| 错误 | 珊瑚红 | #EF4444 | 异常、退步 |
| 背景 | 浅灰白 | #F8FAFC | 页面背景 |
| 卡片 | 纯白 | #FFFFFF | 卡片背景 |
| 文字主色 | 深灰 | #1E293B | 标题、正文 |
| 文字辅色 | 中灰 | #64748B | 描述、次要信息 |

### 1.2 字体规范

| 用途 | 字体 | 大小 | 字重 |
|------|------|------|------|
| 页面标题 | Inter | 24px | 600 |
| 卡片标题 | Inter | 18px | 600 |
| 正文 | Inter | 14px | 400 |
| 辅助文字 | Inter | 12px | 400 |
| 数字/指标 | JetBrains Mono | 24px | 500 |

### 1.3 响应式断点

| 设备 | 断点 | 布局 |
|------|------|------|
| 移动端 | < 768px | 单列、全宽 |
| 平板 | 768px - 1024px | 双列 |
| 桌面 | > 1024px | 多列、侧边栏 |

#### 2.3.1 元音录音模式

```
┌──────────────────────────────────────────────────────────────┐
│  ← 返回                    元音录音                          │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  👤 学生: 李四同学                                          │
│                                                              │
│  ─── 当前元音 ───────────────────────────────────────────   │
│                                                              │
│                      🔴                                      │
│                     "A"                                     │
│                    大声唱出                                  │
│                                                              │
│  ─── 元音进度 ──────────────────────────────────────────   │
│  [●] A  [ ] I  [ ] O  [ ] U  [ ] E                       │
│                                                              │
│                    ╭─────╮                                   │
│                    │ ●●● │   ←  波形动画                     │
│                    ╰─────╯                                   │
│                      3.2s                                    │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ 提示: 请用稳定的气息唱出 "A" 音，保持3-5秒           │   │
│  └──────────────────────────────────────────────────────┘   │
│                                                              │
│              [ 暂停 ]        [ 停止并保存 ]                  │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

**交互说明:**
- 依次引导录制 a/i/o/u/e 五个元音
- 每个元音3-5秒稳定发声
- 自动识别并标记元音类型
- 完成后自动进行嗓音分析

```
│                                                              │

│                                                              │
└──────────────────────────────────────────────────────────────┘
```

**交互说明:**
- 测试前显示历史最佳成绩作为参考
- 倒计时3-2-1后开始录音
- 实时显示已持续时间
- 波形显示音量变化（检测发声是否稳定）
- 自动停止：当音量低于阈值或用户主动停止
- 测试完成后显示：持续时长、气息评分、改进建议

**评分参考:**
```
成年男性: ≥25秒优秀 | 15-25秒良好 | <15秒需加强
成年女性: ≥20秒优秀 | 12-20秒良好 | <12秒需加强
儿童:     ≥15秒优秀 | 8-15秒良好  | <8秒需加强
```
