# VocalDoc 声乐课堂嗓音档案管理系统 - 用户手册

## 1. 系统概述

VocalDoc 是一套专业的声乐教学嗓音档案管理系统，支持教师管理学生、录音录制、嗓音分析、历史对比和AI智能建议。

## 2. 用户角色

### 2.1 管理员 (ADMIN)
- 管理教师账号
- 系统配置
- 数据统计

### 2.2 教师 (TEACHER)
- 管理学生列表
- 查看/分析学生录音
- 录入评语
- 历史对比分析

### 2.3 学生 (STUDENT)
- 录制音频
- 查看分析结果
- 查看历史趋势
- 接收AI建议

## 3. 功能使用

### 3.1 登录
1. 打开系统首页
2. 输入手机号
3. 获取短信验证码
4. 输入验证码登录

### 3.2 录音
1. 进入录音页面
2. 选择录音类型：
   - **元音练习**: a/i/o/u/e 每个3-10秒
   - **声时测试**: 持续发音最长时间
   - **完整曲目**: 演唱完整歌曲
3. 点击录音按钮
4. 录音完成后可回放确认

### 3.3 查看分析
1. 进入档案列表
2. 选择一条录音
3. 查看各项指标：
   - 音高 (F0)
   - 共振峰 (F1-F4)
   - 音量 (dB)
   - Jitter/Shimmer
   - 谐噪比 (HNR)

### 3.4 历史对比
1. 选择两条录音
2. 查看对比结果
3. 了解进步/退步情况

### 3.5 AI建议
1. 查看个人分析结果
2. 获取AI个性化建议
3. 了解训练重点

## 4. 指标参考

### 4.1 正常范围
| 指标 | 正常范围 | 异常参考 |
|------|----------|----------|
| Jitter (local) | < 1.04% | > 1.04% |
| Shimmer (local) | < 3.81% | > 3.81% |
| HNR | > 10 dB | < 10 dB |

### 4.2 声部参考
- 女高音: F1 200-350Hz, F2 2000-2800Hz
- 女低音: F1 180-300Hz, F2 1600-2400Hz
- 男高音: F1 160-260Hz, F2 1400-2200Hz
- 男中音: F1 140-220Hz, F2 1200-2000Hz
- 男低音: F1 100-180Hz, F2 800-1500Hz

## 5. 常见问题

### Q: 录音无法上传？
A: 检查网络连接，确保音频格式为wav/flac/mp3/m4a

### Q: 分析结果为空？
A: 录音时间太短或音频质量不佳，建议重新录制

### Q: 如何联系教师？
A: 联系管理员添加教师微信或手机号
