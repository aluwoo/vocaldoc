# 声乐课堂嗓音档案管理系统 - 开发计划

## 项目概述

- **项目名称**: 声乐课堂嗓音档案管理系统 (VocalDoc)
- **项目类型**: B/S 架构 Web 应用
- **技术栈**: Java (Spring Boot) + Vue 3 + Python (Parselmouth)
- **预计周期**: 3-4 个月

---
